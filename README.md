# 🎬 Cinemystery

Cinemystery is a web-based movie guessing game where players test their movie knowledge by identifying movies from partially hidden titles.

The game presents a masked movie name (for example: `A _ _ M _ L`) and the player must guess the correct movie title.

The project focuses on creating a **clean cinematic UI, interactive gameplay, and a fun guessing challenge** using only frontend technologies.

---

# 🚀 Features

## 🎮 Game Modes

### Single Player
- Guess movie titles from hidden letters
- Score points based on correct guesses
- Use hints when stuck

### Local Multiplayer
- Two players play on the same device
- Turn-based gameplay
- Score tracking for both players

---

# 🎯 Gameplay

1. A movie title appears with hidden letters.
2. The player enters a guess.
3. The system checks if the guess is correct.
4. Points are awarded for correct answers.
5. Hints can be used to reveal clues.


---

# 🧠 Hint System

Players can reveal hints such as:

- Reveal a random letter
- Show movie release year
- Show movie genre
- Show a short description

Hints reduce the final score.
